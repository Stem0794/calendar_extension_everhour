# Weekly Calendar Project Tracker Chrome Extension

## Install
1. Download and unzip this folder
2. Go to `chrome://extensions` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select this unzipped folder

## Use
- Open Google Calendar in Week View
- Click the extension icon
- Manage projects (color, keywords, etc) in Projects tab
- Assign meetings in Summary tab, filter by day or week
- Hours tab = see project totals

## Data is saved in Chrome storage.
